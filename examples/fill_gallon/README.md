# Fill Gallon

A sample command-line application using yeslist_fill_gallon library.

## Requirements

  To run the source code project please see: https://dart.dev/get-dart

## Run

    - from source code:

    dart run ./bin/fill_gallon.dart

    - from windows:

    ./bin/fill_gallon.exe 

    - from linux:

    ./bin/fill_gallon


    example: fill_gallon 1
  
    see '--help' option
